    Simple Pokedex

### Technologie Used
  - React
  - Typescript
  - Styled Components
  - TailwindCSS

### Requirements

* [NodeJS](https://nodejs.org/en/)  Please use LTS version (NVM is recommended if you use MacOS)
* [Git](https://git-scm.com/)

### Install yarn

```shell
npm i -g yarn
```

## Quick start

1. **Unzip repository**

```shell
unzip pokedex.zip
```

2.  Navigate to Root
  
```shell
cd pokedex
```

3.  **Install Packages**

```shell
yarn install
```

4.  **Run Build process**

```shell
yarn build
```

5. **Serve**
You can use ViteJS
```shell
yarn serve
```
